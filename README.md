# Testimonials Grid Section

This is a solution to the [Testimonials grid section challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/testimonials-grid-section-Nnw6J7Ma7).

## Overview

The challenge was to build a testimonials grid section that matches the provided design. It works on multiple screen sizes, displaying a grid layout on desktop and a stacked layout on mobile.
